Nodes for translating, rotating, and scaling 2D and 3D graphics.

They can be used with the [vuo.layer](vuo-nodeset://vuo.layer) node set for drawing 2D layers or the [vuo.scene](vuo-nodeset://vuo.scene) node set for drawing 3D scenes.

**Translation** changes the position. **Rotation** changes the angle. **Scaling** changes the size.

When the transformation is applied to a layer or object, the layer or object is first rotated, then scaled, then translated.
